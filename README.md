# 漫跃短视频推荐系统

[![项目演示视频](https://img.shields.io/badge/演示视频-Bilibili-00A1D6?style=for-the-badge&logo=bilibili)](https://www.bilibili.com/video/BV1WeBjB2Ez7/?spm_id_from=333.337.search-card.all.click&vd_source=9a55113f83de31fa8eca0192b4c47404)

## 📋 项目简介

漫跃短视频推荐系统是一个集成了小红书和B站功能的综合性短视频平台。系统采用协同过滤算法提供个性化视频推荐，支持丰富的社交互动功能，包括直播、私信、视频通话等，并整合了AI智能助手功能。

## ✨ 核心特性

### 🎯 视频功能
- **发布视频**：支持用户上传和发布短视频
- **观看体验**：观看视频、查看观看历史、收藏视频
- **互动功能**：评论视频、发送弹幕
- **实时数据**：实时更新同时观看人数、实时收发弹幕
- **智能审核**：使用阿里云视频点播服务进行自动视频审核和发布

### 🔍 智能搜索
- **Elasticsearch集成**：提供高效的内容搜索体验
- **个性化排序**：随机排序算法（不同用户看到不同顺序）
- **关键词高亮**：搜索结果中关键词高亮显示

### 🤖 推荐系统
- **协同过滤算法**：
  - 基于用户的视频推荐
  - 基于视频的视频推荐（内容相似度推荐）

### 💬 社交互动
- **关注订阅**：关注用户功能，粉丝接收更新通知
- **实时通信**：WebRTC实现的直播功能和视频通话
- **私信系统**：用户间实时私信交流

### 🛠️ 技术优化
- **弹幕优化**：WebSocket技术配合消息队列优化弹幕体验
- **数据持久化**：定时任务和多线程技术处理点赞、收藏、观看次数等数据
- **系统稳定**：AOP实现登录鉴权和接口限流
- **AI集成**：对接SpringAI和知识库系统
